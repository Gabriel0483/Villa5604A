# **App Name**: LeaseLink

## Core Features:

- Tenant Profile Management: Create, view, and edit detailed profiles for each tenant, including contact information and lease terms.
- Rent Payment Tracking: Record and monitor monthly rent payments, view payment history, and track outstanding balances and due dates.
- Utility Bill Allocator: Input shared utility bills (Wifi, Water, Electricity) and automatically calculate pro-rata shares for each tenant based on user-defined criteria.
- Repair Request & Status: Provide a portal for tenants to submit repair requests and allow the head tenant to manage and update the status of each request.
- Pro-Rata Allocation Tool: An AI tool that suggests optimal pro-rata sharing methodologies for utility bills based on input parameters such as tenant count, room size, or estimated individual usage.

## Style Guidelines:

- Primary color: A modern, trustworthy blue-green (#1F99B2) for action elements and headings. This conveys reliability and efficiency.
- Background color: A very light, desaturated blue-green (#ECF6F8) providing a clean, bright, and professional canvas that is easy on the eyes.
- Accent color: A fresh and complementary green (#5EDB9F), providing clear contrast for interactive elements or status indicators without overwhelming the primary palette.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective readability, ideal for data-heavy management interfaces.
- Utilize a consistent set of clean, line-based icons that visually communicate functionality clearly, supporting a minimalist and efficient user experience.
- Implement a responsive, card-based layout for tenant profiles and bill summaries, allowing for easy scanning of information on various screen sizes. Data tables should be organized for quick comprehension of rents and utilities.
- Incorporate subtle, functional animations for transitions between sections, form submissions, and status updates to enhance user feedback and perceived performance without distraction.
